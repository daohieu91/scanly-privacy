# scanly-privacy

Public privacy policy for [Scanly](https://play.google.com/store/apps/details?id=com.essenty.scanly), an Android PDF scanner app.

**Live policy:** https://daohieu91.github.io/scanly-privacy/

## Editing

The policy is a single `index.html` file. Edit it, bump the "Last updated" date at the top, commit, and push. GitHub Pages republishes within ~60 seconds.

Material changes (added data collection, jurisdiction change, etc.) should also be announced in the app's Settings screen so users see the notice.
